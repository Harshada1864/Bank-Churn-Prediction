Bank Customer Churn Prediction Visualizations

This page shows the key visualizations from my churn prediction project. I analyzed a bank dataset to predict which customers might leave, using data analysis and machine learning. These plots are easy to understand and show my findings.

Exploratory Data Analysis

Churn Distribution: This shows how many customers stayed (0) vs. left (1). Most customers stay, but about 20% churn, so the data is imbalanced.

Correlation Heatmap: This shows how features like age and balance relate to each other and churn. Strong correlations (red or blue) help identify important factors.

Age vs. Churn: This box plot compares the age of customers who stayed vs. left. Older customers seem more likely to churn.

Balance vs. Churn: This shows how account balance relates to churn. Customers with higher balances may have different churn patterns.

Model Performance

ROC Curve - Logistic Regression: This shows how well Logistic Regression predicts churn. The AUC score indicates its performance (higher is better).

ROC Curve - Random Forest: This plots the Random Forest models performance. It usually does better than Logistic Regression.

ROC Curve - XGBoost: This shows XGBoosts performance, which is often the best due to its ability to handle complex patterns.

Explainable AI (SHAP)

SHAP Feature Importance: This plot shows which features (like age or products_number) most influence churn predictions. Red means higher values increase churn risk.

SHAP Dependence Plot: This shows how the top feature (e.g., age) affects churn predictions. It helps explain why certain customers are at risk.
